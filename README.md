ÖDEV 7

Güneş sistemindeki tüm gezegenleri listeleyen bir Gezegen sınıfı tasarlayın. Her gezegen, Güneş'e olan
sıralama numarasını, uzaklığı, yarıçapı, dönme süresini içeren bilgileri içermelidir (kesin veriler gerekli
değil). Ödevinizi Github reposu olarak iletin.
Ödev adımları:
1. Enum tipinde Gezegen sınıfını oluşturun. (4 puan)
2. Gezegen özelliklerini atayın. (3 puan)
3. Gezegen özelliklerini ekrana yazdıran uygulamayı tamamlayın. (3 Puan)
